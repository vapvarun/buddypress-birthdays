## Overview

Transform your BuddyPress community with the Birthday Widget! Know the upcoming birthdays of your site's members and help them celebrate their special day. A little effort to greet community members keeps the saying alive: "community that celebrates together stays together".

## Features

- **Smart Birthday Display** — Show birthdays for all members, friends, or followers with flexible time ranges
- **Today's Birthdays** — Special highlighting with optional confetti animation
- **Age Display** — Customizable "Turning X" format
- **Automatic Notifications** — Birthday email greetings, activity feed auto-posts, BuddyPress notifications
- **Admin Summary Email** — Daily birthday digest for site administrators
- **Zodiac Signs** — Optional zodiac sign display
- **Responsive Design** — Mobile-first layout with smooth animations
- **Shortcode Support** — `[bp_birthdays]` for flexible placement
- **Performance Optimized** — 30-minute smart caching, conditional asset loading
- **WCAG 2.1 Compliant** — Keyboard navigation, screen reader support, reduced motion

## Installation

1. Upload the `buddypress-birthdays` folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **Appearance > Widgets** and add the "BuddyPress Birthdays" widget
4. Or use the shortcode `[bp_birthdays]` in any page or post

## Shortcode

```
[bp_birthdays range="monthly" display="friends" show_age="yes" limit="10"]
```

| Parameter | Options | Default |
|-----------|---------|---------|
| `range` | `weekly`, `monthly`, `unlimited` | `monthly` |
| `display` | `all`, `friends`, `followers` | `all` |
| `show_age` | `yes`, `no` | `no` |
| `limit` | Any number | `10` |
